NIGHTMARE HOSPITAL
Close

Nightmare Hopsital is a Java console game created over the course of a weekend.

Using Git, I worked with a partner to build a console game in Java using OOP. It highlights my knowledge of Abstraction, Polymorphism, Inheritance and Encapsulation. This project was built using TDD.

The most obvious challenge was completing this project over the course of only 30 hours. Additionally, this was my first use of interfaces, final and abstract classes as well as Java constants. Working remotely from my partner required a better understanding of git source control, which I acquired.

This was the first project in which I fully utilized, and gained an immense appreciation for, test-driven development. Testing first required thinking through the intended outcomes of methods which paved a roadmap for the ultimate direction of the project.

Check it out on GitHub!

VIRTUAL PET
Close

Virtual Pet is a tamagotchi-like Java console application. You must bath, feed and play with your pets to keep their health levels up.

This application was developed in a team of 5 over the course of 5, 3-day sprints. It required the use of git, TDD with JUnit, Java and OOP in an agile environment.

We used while loops, for loops and switch statements, gaining an understanding of when to use each type of control flow statement. Additional challenges including learning how and when to use a HashMap, and how to iterate over it.

Check it out on GitHub!
